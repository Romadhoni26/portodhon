About Romadhoni.

Selama studi saya, saya telah terlibat dalam berbagai proyek, termasuk membuat website dengan aplikasi yang suda di siapkan seperti, visual studio code dan mysql. Di luar studi, saya tertarik dengan membuat web dan sering menghabiskan waktu untuk membuat web. Saya bersemangat untuk terus belajar dan mengeksplorasi berbagai aspek dari rekayasa perangkat lunak serta berkontribusi pada proyek-proyek yang menantang.
